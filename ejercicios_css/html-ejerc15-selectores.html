    Añadido dicho "id" aqui, no puede ser utilizado en otra parte.

  3. Todos los elementos (*)

    4. Selector que selecciona a todos los elementos del documento para ponerles el mismo estilo.

    Ejemplo: En este documento podríamos haber añadido un selector * para que todo el documento tenga un mismo estilo; por ejemplo, un tipo de letra determinada.

  4. Varios elementos (p, em, h1, h2)

    5. El selector "p" sirve para organizar el documento en párrafos.

    Ejemplo: cada párrafo escrito en este documento esta dentro de un selector "p" (uno para abrir y otro para cerrar).

    El selector "em" es utilizado para escribir en cursiva.

    Ejemplo:Añadimos una etiqueta "em" y así esto, está escrito en letra cursiva.

    Los selectores "h1" y "h2" sirven para establecer los títulos.

    Ejemplo: Vamos a añadir dichas etiquetas:

    Este primero es marcado por un h1

    (por lo que es más grande, ya que se utilizaría como para el título principal)

    Y este segundo, por un h2

    (este algo más pequeño porque es más utilizado en subtítulos)

  5. Elementos contenidos en otros elementos (ol li)

    6. La etiqueta "ol" es utilizada para establecer un lista ordenada de distintos elementos. Dichos elementos deben encontrarse entre un selector "li" para que el sistema se de cuenta de que dichos elementos forman parte de la lista. Además, estos elementos "li" pueden tener su propio estilo.

    Ejemplo:

    1. Debemos poner la etiqueta "ol".

    2. Debemos añadir la etiqueta "li", tanto para introducir como para cerrar, cada elemento que queramos añadir a la lista. Este además tiene su propio estilo.

    3. Cerramos la etiqueta "ol" una vez terminada la lista

  6. Sobrevuelo (p:hover)

Con este selctor podemos dar un estilo concreto a una parte del documento pero, dicho estilo, solo se aprecia cuando se pasa el ratón por encima.
